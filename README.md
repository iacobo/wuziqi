## About

A simple material design Wuziqi game (aka [Gomoku, 5-in-a-row, Caro](https://en.wikipedia.org/wiki/Gomoku)).

Completely FOSS, no ads, no trackers, nothing unnecessary. Just a clean, offline, 2-player boardgame app.
